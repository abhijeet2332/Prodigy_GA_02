# Prodigy_GA_02
# Image Generation with Pre-trained Models

## Overview
This project generates images from text prompts using pre-trained generative models such as **DALL·E-mini** or **Stable Diffusion**. The model interprets natural language descriptions and produces corresponding visual outputs.

## Features
- Text-to-image generation
- Uses pre-trained generative AI models
- Supports custom text prompts
- Generates high-quality images automatically

## Technologies Used
- Python
- Hugging Face Diffusers / Transformers
- PyTorch
- Stable Diffusion or DALL·E-mini

## Project Workflow
1. Load the pre-trained image generation model.
2. Provide a text prompt describing the desired image.
3. The model processes the prompt and generates an image.
4. The generated image is displayed or saved.

## Usage
Run the script and enter a text prompt to generate an image based on the description.

## Example Prompt
"A futuristic city at sunset with flying cars and neon lights."

## Output
The model generates an image that visually represents the given text prompt.
